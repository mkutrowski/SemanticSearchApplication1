# Nazwa Projektu 
SemanticSearchApplication
## Opis 
Aplikacja:
- pobiera plika w formacie pdf lub docx.
- dzieli na fragmenty i je indeksuje 
- wyszukuje najlepiej pasujące fragmenty zgodnie z wybraną metryką

## Uruchomienie

  - uvicorn app.main:app --reload
## Dostęp do GUI
  - W przyglądarce wpisać http://127.0.0.1:8000

## Użycie:
  - Pobierz plik PDF lub DOCX
  - Podziel i zaindeksuj wybrny dokument
  - Wyszukaj najlepiej pasujące fragmenty
